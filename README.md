# Night Type - 夜の性格診断

32種類の夜の性格診断で、あなたに最適な夜の過ごし方を発見。

## 🌙 概要

Night Typeは、夜間の行動パターンと性格特性を分析し、個人に最適化された夜のライフスタイルを提案するWeb診断アプリケーションです。

## ✨ 特徴

- **32種類の詳細分析**: 男女各16タイプの夜のキャラクター
- **科学的アプローチ**: 性格分析に基づいた診断システム
- **簡単診断**: わずか24問・3分で完了
- **完全無料**: 登録不要で利用可能
- **レスポンシブ対応**: あらゆるデバイスで最適な体験

## 🚀 技術スタック

- **フレームワーク**: Next.js 16.1.1
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS 4.0
- **アニメーション**: Framer Motion
- **デプロイ**: Vercel
- **SEO最適化**: 構造化データ、OGP対応

## 📱 主要機能

### 診断システム
- 24問の詳細な質問
- 4軸分析（AP, RF, TE, NC）
- リアルタイム結果表示

### キャラクター分析
- 32種類の詳細なキャラクタープロファイル
- 個別の性格分析とアドバイス
- カスタムキャラクターページ

### ユーザー体験
- 直感的なUI/UX
- スムーズなアニメーション
- ソーシャルシェア機能
